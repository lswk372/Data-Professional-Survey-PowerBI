# Data Professional Survey Dashboard

## Overview

This project analyzes survey responses from 630 data professionals to better understand trends in the data industry, including job roles, salary ranges, programming language preferences, career-switching patterns, and workplace satisfaction.

The goal of this project was to build an interactive Power BI dashboard that turns raw survey data into clear and easy-to-understand insights.

## Project Objectives

The main objectives of this project were to clean, analyze, and visualize survey data in order to identify key trends among data professionals. I focused on creating a dashboard that could help users quickly understand common job roles, salary patterns, preferred technical skills, and satisfaction levels in the data field.

## Data Preparation

The original dataset included survey responses about job title, salary range, preferred programming language, country, education, career-switching experience, and satisfaction levels.

I cleaned and transformed the dataset in Power Query by removing unnecessary columns, renaming fields for readability, standardizing inconsistent survey responses, creating salary estimate calculations from salary ranges, and preparing the data for dashboard-level analysis.

## Dashboard Features

The final Power BI dashboard includes KPI cards, slicers, bar charts, donut charts, and gauges to summarize the survey results. The dashboard allows users to explore total respondents, average age, salary by job title, favorite programming languages, difficulty breaking into data, salary satisfaction, work-life balance satisfaction, and country-level filtering.

## Result of Analysis

The dashboard showed that Data Analyst was the most common role among respondents, while Python was the most preferred programming language. Many respondents had switched careers into data, showing that the field attracts people from different backgrounds. The analysis also found that salary satisfaction was lower than work-life balance satisfaction, suggesting that compensation was a key concern. Overall, most respondents felt that breaking into data was neither easy nor difficult, meaning the field is accessible but still requires preparation and skill development.

## Key Insights

- Data Analyst was the most common role among survey respondents.
- Python was the most preferred programming language.
- Many respondents had switched careers into data.
- Salary satisfaction was lower than work-life balance satisfaction.
- Most respondents described breaking into data as neither easy nor difficult.
